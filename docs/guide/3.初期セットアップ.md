## 3. 初期セットアップ

- 3.1 事前準備
- 3.2 プロジェクト作成
- 3.3 開発サーバの起動

---

### 3.1 事前準備

本講座では、Node.js 環境上で React SPA を開発します。  
アプリを動作させる場合は、事前に次のツールが利用可能であることを確認してください。

#### Node.js

Node.js は、React の開発環境やビルドツールを動作させるために必要です。

バージョン確認方法（Node.js 22 以降推奨）:
```bash
node -v
```
Node.js がインストールされていない、またはバージョンが古い場合は、公式サイトの案内に従ってインストールしてください。

- Node.js 公式サイト
https://nodejs.org/


#### pnpm

pnpm は、Node.js 用のパッケージマネージャです。本講座では、依存関係管理とコマンド実行に pnpm を使用します。

バージョン確認方法:
```bash
pnpm -v
```

pnpm が利用できない場合は、公式ドキュメントの手順に従ってセットアップしてください。

- pnpm 公式ドキュメント
https://pnpm.io/installation

※ npm や yarn でも動作自体は可能ですが、本講座では pnpm を前提 に進めます。

### 3.2 React Router Frameworkモードの作成

以下のコマンドを実行し、React Router（Framework モード）のプロジェクトを作成できます。
ここではプロジェクト名を react-sample とします。

```bash
pnpm create react-router@7.10.1 react-sample
```

コマンドを実行すると、いくつか設定を聞かれる場合がありますが、基本的には デフォルトのまま（Enterキーを押す） で進めて問題ありません。

### 3.3 開発サーバの起動

プロジェクトの作成が完了したら、作成されたディレクトリに移動して開発サーバを起動します。
```bash
cd react-demo
pnpm dev
```

コマンド実行後、ターミナルに表示された URL（例: http://localhost:5173）へアクセスし、初期画面が表示されればセットアップは完了です。
